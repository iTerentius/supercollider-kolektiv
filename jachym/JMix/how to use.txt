stableVersion 0.12

Put stableVersion/JMix folder to your Platform.systemExtensionDir and recompile the class library (Ctr+Shift+L). 

To build new efx just save to /efx new file in form like example:


SynthDef(\Efx_LPF, { |bus, freq|
	ReplaceOut.ar(bus, LPF.ar(In.ar(bus, 2),freq))
	},
	metadata: (
		specs: (
			freq: ControlSpec(50, 20000.0, \lin, 10, 400)
		)
	)
)



// name of file have to be same like name of SynthDef (Efx_LPF.scd -> \Efx_LPF)

// output of synthdef have to named like |bus| or |out|

// metadata are for preset of controlers (minVal, maxVal, \lin, step of scroll, defaultVal)
